## Image Quality Standards

- **Resolution:** 300 DPI for print, 72 DPI for web
- **File format:** SVG (scalable vector graphics) preferred for diagrams; PNG for photo-realistic elements
- **Color palette:** Accessible to colorblind viewers (avoid red-green distinctions alone)
- **Label readability:** Minimum 12pt font for body text, 14pt for figure titles
- **Molecular accuracy:** Use standard biochemistry notation (ATP = adenosine triphosphate with ribose + three phosphates), conformational changes illustrated clearly
